# Type Speed Test

A typing speed test with live WPM, accuracy, and error tracking. Dark and light mode included.

**Live:** https://reneebe.github.io/type-speed-test/

## Features

- Live WPM, accuracy, time, and error stats
- Character-by-character highlighting (green/red)
- Blinking cursor tracking position
- Dark and light mode toggle (persisted in localStorage)
- 10 randomized passages

## Stack

- React + TypeScript
- Vite

## Development

```bash
npm install
npm run dev
```
